Software Stack

Stellarium

Real-time sky simulation software used for celestial mapping, object tracking, and observation planning.

DeepSkyStacker

Stacking software used to combine multiple exposures, improving signal-to-noise ratio in deep-sky imaging.

Adobe Lightroom

Post-processing software for refining astrophotography, adjusting exposure, clarity, and color balance.
